Self-hosting là gì?

(hay còn gọi là tự host, tự lưu trữ)

Self-hosting là việc bạn tự chạy và quản lý các dịch vụ, ứng dụng, website, lưu trữ dữ liệu… trên máy chủ do chính bạn kiểm soát, thay vì dùng dịch vụ của các công ty lớn như Google, Dropbox, iCloud, YouTube, Spotify, v.v.

Nói đơn giản: thay vì "thuê" dịch vụ của người khác → bạn tự làm "chủ nhà" luôn.

So sánh nhanh để dễ hình dung

Tiêu chí Dùng dịch vụ bên thứ 3 (Cloud) Self-hosting
Ví dụ Google Drive, Gmail, Netflix, Spotify Tự cài Nextcloud, Immich, Jellyfin, Vaultwarden
Ai quản lý server Công ty (Google, Microsoft, Dropbox) Chính bạn (hoặc thuê VPS nhưng vẫn bạn quản lý)
Quyền kiểm soát dữ liệu Thấp – họ có thể đọc/xem/xóa tài khoản Cao – dữ liệu nằm trên máy bạn
Chi phí dài hạn Thường đắt dần (subscription) Ban đầu tốn tiền mua phần cứng/VPS, sau rẻ hơn
Độ khó Dễ, chỉ cần đăng ký Khó hơn, cần học cài đặt, bảo trì
Bảo mật & riêng tư Phụ thuộc chính sách công ty Tốt hơn nếu bạn biết cấu hình đúng
Nếu công ty "tắt" Mất hết dữ liệu/dịch vụ Vẫn chạy bình thường (trừ khi bạn tắt)

Các loại self-hosting phổ biến hiện nay (2026)

  • Tại nhà (Home server / Homelab)
    Dùng máy tính cũ, Mini PC, Raspberry Pi, hoặc NAS (Synology, QNAP, tự build)
    Chạy 24/7 trong nhà
    Phổ biến nhất với người mới
  • Trên VPS/Cloud server thuê
    Thuê máy ảo từ Hetzner, Contabo, DigitalOcean, Vultr, Oracle Cloud (free tier)…
    Bạn vẫn tự cài mọi thứ → vẫn tính là self-hosting
    Ưu điểm: luôn online, IP công cộng, tốc độ tốt hơn nhà nhiều
  • Hybrid (kết hợp)
    Lưu trữ dữ liệu nặng ở nhà → truy cập từ xa qua VPN/Cloudflare Tunnel/Tailscale
    Dùng VPS làm "cửa ngõ" bảo mật

Những thứ người ta hay self-host nhất (2025–2026)

  • Lưu trữ file & ảnh cá nhân → Nextcloud, Immich, PhotoPrism
  • Xem phim/series → Jellyfin, Plex, Stremio + Torrent
  • Quản lý mật khẩu → Vaultwarden (Bitwarden tự host)
  • Tải phim/nhạc/truyện tự động → *arr suite (Radarr, Sonarr, Prowlarr, Lidarr…)
  • Nhật ký & ghi chú → Obsidian + Syncthing hoặc Logseq + self-host
  • Chat riêng tư → Matrix + Element, XMPP
  • Git/private code → Forgejo, Gitea
  • Trang web/blog cá nhân → WordPress, Ghost, Astro… tự host

Tóm lại:

Self-hosting = Tự do + Riêng tư + Học hỏi nhiều,
đổi lại là tốn thời gian + công sức + đôi khi tốn thêm tiền điện/đồ.
